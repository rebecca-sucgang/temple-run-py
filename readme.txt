Project Title: Temple Run Lite

Description:
This is Temple Run Lite, a Python game based on "Temple Run", using object-oriented programming and top-down design. 

Requirements:
Python 3.11.0
cmu_graphics
random
PIL
json

To Use:
- Run game.py
- Must have the latest version of CMU Graphics installed
  - CMU Graphics supports Windows (10+), macOS (10.13+), and Python (3.6-3.11)
