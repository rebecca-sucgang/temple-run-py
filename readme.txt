Project Title: Temple Run Lite

Description:
This is Temple Run Lite, a Python game based on "Temple Run", using object-oriented programming, top-down code design, Model-View-Controller architecture, and structured JSON data handling to store and manage player metrics securely. This game includes a random maze-generation feature using the Breadth-First Search algorithm and interactive UI and sprite animations using CMU Graphics.

Requirements:
Python 3.11.0
cmu_graphics
random
PIL
json

To Use:
- Run game.py
- Must have the latest version of CMU Graphics installed
  - CMU Graphics supports Windows (10+), macOS (10.13+), and Python (3.6-3.11)
